## Installation
1. Create the virtual environment
    ```shell
    python -m venv .venv
    ```
2. Activate the virtual environment
   ```shell
   . .venv/bin/activate
   ```

3. Install dependencies
   ```shell
   pip install -r requirements.txt
   ```
